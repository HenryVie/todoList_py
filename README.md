# 📝 To-Do List CLI Application (Python)

A command-line based task management application developed in Python as a final project for CS50’s Introduction to Programming with Python.

This application allows users to manage tasks and custom lists directly from the terminal, focusing on simplicity, functionality, and structured program design.

## 🚀 Features

### ✅ Task Management
- Add new tasks with optional due dates
- Mark tasks as completed
- Edit task name or due date
- Delete tasks
- View all tasks with completion status

### 📋 List Management
- Create custom lists
- Add or remove items from lists
- View all lists and their contents
- Delete lists

### 🧠 Additional Highlights
- Menu-driven CLI interface
- Input validation with error handling
- Clear data structure using Python dictionaries and lists

## 🛠️ Technologies Used
- Python 3
- Command-Line Interface (CLI)
